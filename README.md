Installing
----------

Preferred way to install is with [Composer](https://getcomposer.org/).

Just add

    "require": {
      "tansoft/baselib": "1.0.*"
    }

in your projects composer.json.

Client usage:
-------------
```php
require('vendor/autoload.php');

```
